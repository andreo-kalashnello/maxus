# MAXAS — Профессиональное оборудование для маникюра

## 📋 Описание проекта

**MAXAS** — официальный сайт европейского бренда профессионального оборудования для маникюра и ногтевого сервиса.

Сайт разработан на чистой верстке (HTML/CSS/JS) и подготовлен к переносу на CMS WordPress. Проект содержит полный каталог продукции, информацию о компании, решения и технологии, B2B-партнёрство и раздел инструкций.

### 🎯 Ключевые разделы

- **Главная** (`index.html`) — обзор бренда, философия и ключевые принципы
- **Продукция** (`pages/products.html`) — каталог оборудования и комплектаций
- **Решения и технологии** (`pages/solutions.html`) — направления и технологические подходы
- **B2B сотрудничество** (`pages/b2b.html`) — информация и форма для партнёрства
- **Инструкции** (`pages/instructions.html`) — руководства по использованию
- **Контакты** (`pages/contacts.html`) — форма обратной связи и реквизиты
- **О компании** (`pages/about.html`) — история и подробная информация

## 🎨 CSS Архитектура

### Структура и scope-система

Все стили **модульны и безопасны**:

- ✅ **Нет конфликтов**: Страницы используют body-классы (`.products-page`, `.contacts-page` и т.д.)
- ✅ **Нет !important**: Селекторы приоритизированы иерархией
- ✅ **Responsive-first**: Все брейкпоинты определены (1024px, 768px, 480px)

### Переменные CSS (global.css)

```css
:root {
	--primary: #A7626C;           /* Основной розовый */
	--accent: #D8C5CC;            /* Акцент */
	--text-dark: #2B2B2B;         /* Тёмный текст */
	--text-light: #FFFFFF;        /* Светлый текст */
	--border: #E8E8E8;            /* Граница */
}
```

## 📝 Подготовка к WordPress

3. **Компоненты**: Переиспользуй существующие классы
